# Настройка сложности клешни (по режимам)

Этот документ описывает, как менять параметры сложности захвата в игре для режимов:
- `NORMAL`
- `RESCUE`
- `FIND_ANIMAL`
- `CATCH_CAT`

Источник параметров: `core/src/main/java/ru/captainxsander/GameTuning.java`.
Применение параметров: `core/src/main/java/ru/captainxsander/MainGame.java` -> `GameSessionSettings` -> `Claw`.

---

## 1) Какие параметры формируют опыт игрока

Для каждого режима есть набор:

- `*_BASE_SLIP_CHANCE`  
  Шанс срыва игрушки при подъёме.

- `*_CLAW_DROP_BASE_CHANCE`  
  Базовый шанс выпадения в пути к лотку.

- `*_CLAW_DROP_MIN_CHANCE`  
  Минимальный порог выпадения в пути (не даёт шансу стать слишком маленьким).

- `*_BASE_FAKE_GRAB_CHANCE`  
  Ложный захват (когда кажется, что схватили, но игрушка не доезжает).

- `*_CLAW_GRAB_X_MARGIN`  
  Ширина «безопасной» зоны по X внутри пальцев клешни.
  - Больше значение = сложнее попасть в валидную зону.
  - Меньше значение = легче захватить.

- `*_CLAW_CATCH_CHANCE_MULT`  
  Множитель базового шанса захвата `1 - catchDifficulty`.
  - `< 1.0` усложняет первичный захват.
  - `> 1.0` упрощает первичный захват.

Дополнительно (общая логика, не по режимам):
- `CLAW_DROP_DIFFICULTY_MULT`, `SLIP_DIFFICULTY_MULT`, `FAKE_GRAB_DIFFICULTY_MULT` усиливают влияние сложности игрушки (`catchDifficulty`).

---

## 2) Что и как крутить в первую очередь

### Если игрок «вообще не может ничего поднять»
1. Уменьшайте `*_CLAW_GRAB_X_MARGIN`.
2. Повышайте `*_CLAW_CATCH_CHANCE_MULT`.
3. Только потом снижайте `*_BASE_FAKE_GRAB_CHANCE`.

### Если игрушка часто берётся, но не доезжает
1. Снижайте `*_CLAW_DROP_BASE_CHANCE`.
2. Снижайте `*_BASE_SLIP_CHANCE`.
3. Проверьте `*_CLAW_DROP_MIN_CHANCE` — он может держать слишком высокий нижний порог выпадения.

### Если игра слишком «рандомная»
- Одновременно снижайте: `*_BASE_SLIP_CHANCE`, `*_CLAW_DROP_BASE_CHANCE`, `*_BASE_FAKE_GRAB_CHANCE`.
- Держите `*_CLAW_CATCH_CHANCE_MULT` ближе к `1.0`.

## 3) Готовые пресеты (лёгкий/средний/тяжёлый)

Ниже — безопасные стартовые диапазоны. Берите как шаблон и проверяйте в игре 10–20 попыток подряд.

## Лёгкий опыт
- `BASE_SLIP_CHANCE`: `0.12 .. 0.18`
- `CLAW_DROP_BASE_CHANCE`: `0.14 .. 0.20`
- `CLAW_DROP_MIN_CHANCE`: `0.04 .. 0.07`
- `BASE_FAKE_GRAB_CHANCE`: `0.08 .. 0.14`
- `CLAW_GRAB_X_MARGIN`: `0.05 .. 0.07`
- `CLAW_CATCH_CHANCE_MULT`: `1.05 .. 1.15`

Ощущение: игрок чаще поднимает и довозит, меньше «обидных» срывов.

## Средний опыт
- `BASE_SLIP_CHANCE`: `0.19 .. 0.24`
- `CLAW_DROP_BASE_CHANCE`: `0.21 .. 0.27`
- `CLAW_DROP_MIN_CHANCE`: `0.07 .. 0.10`
- `BASE_FAKE_GRAB_CHANCE`: `0.15 .. 0.22`
- `CLAW_GRAB_X_MARGIN`: `0.08 .. 0.10`
- `CLAW_CATCH_CHANCE_MULT`: `0.96 .. 1.04`

Ощущение: баланс между навыком и риском неудачи.

## Тяжёлый опыт
- `BASE_SLIP_CHANCE`: `0.25 .. 0.34`
- `CLAW_DROP_BASE_CHANCE`: `0.28 .. 0.36`
- `CLAW_DROP_MIN_CHANCE`: `0.10 .. 0.14`
- `BASE_FAKE_GRAB_CHANCE`: `0.23 .. 0.32`
- `CLAW_GRAB_X_MARGIN`: `0.11 .. 0.15`
- `CLAW_CATCH_CHANCE_MULT`: `0.84 .. 0.95`

Ощущение: высокая требовательность к точности, частые срывы и выпадения.

---

## 4) Важные сочетания параметров

### Сочетание A: «честная сложность» (навык важнее рандома)
- Слегка увеличиваем `CLAW_GRAB_X_MARGIN`.
- Держим `BASE_FAKE_GRAB_CHANCE` умеренным.
- Не поднимаем слишком высоко `CLAW_DROP_BASE_CHANCE`.

Результат: сложность через точность позиционирования, а не через случайные потери.

### Сочетание B: «драматичный автомат»
- Умеренная зона захвата (`CLAW_GRAB_X_MARGIN` средняя).
- Повышенные `BASE_FAKE_GRAB_CHANCE` + `CLAW_DROP_BASE_CHANCE`.

Результат: больше эмоциональных «почти выиграл» моментов.

### Сочетание C: «обучающий / детский»
- Низкие `BASE_SLIP_CHANCE`, `CLAW_DROP_BASE_CHANCE`, `BASE_FAKE_GRAB_CHANCE`.
- Низкий `CLAW_GRAB_X_MARGIN`.
- `CLAW_CATCH_CHANCE_MULT > 1.0`.

Результат: частые успехи, меньше фрустрации.

---

## 5) Рекомендуемый процесс тюнинга

1. Выберите целевой UX (лёгкий/средний/тяжёлый).
2. Поставьте пресет из раздела 3.
3. Сначала правьте только 2 параметра:
   - `*_CLAW_GRAB_X_MARGIN`
   - `*_CLAW_DROP_BASE_CHANCE`
4. Потом доводите:
   - `*_BASE_SLIP_CHANCE`
   - `*_BASE_FAKE_GRAB_CHANCE`
   - `*_CLAW_CATCH_CHANCE_MULT`
5. Проверяйте каждый режим отдельно (у них разные игровые задачи).

---

## 6) Про `catchDifficulty` у игрушек

Да, у игрушек есть собственная сложность: `catchDifficulty`.
Она участвует в:
- вероятности первичного захвата (`1 - catchDifficulty`),
- срыве при подъёме,
- выпадении в пути,
- ложном захвате,
- скоринге выбора «лучшей» игрушки под клешнёй.

Поэтому при изменении режимных параметров всегда учитывайте состав пула игрушек в режиме.
